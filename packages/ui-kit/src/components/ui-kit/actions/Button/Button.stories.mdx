# Button
Buttons can be used to show the user a choice of options/actions and assign to them clear hierarchy.
The target page to be accessed or the action to be triggered is shown on the buttons by means of a label or else by an icon, or by a combination of the two. 
All buttons provide visual feedback when clicked or tapped in the form of a response effect.

## Variants
### Primary Button
The filled-out primary button is used for the most important action on a page or in a view. The buttons can also contain an icon and the minimum size is 184px while it’s height is set to 48px.

### Secondary Button
The secondary button with a thin 2px frame is used for subordinate actions. 
The secondary button is used either white or transparent depending on the contrast. They can also contain an icon and the minimum size is 184px while it’s height is set to 48px.
