# Grid System

Our grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive. Below is an example and an in-depth look at how the grid comes together.

## Container

Containers provide a means to center and horizontally pad your site’s contents. Use <b>Container</b> for a responsive pixel width.

<Canvas>
  <Container>
    <Row>
      <Col>
        <div>1 of 1</div>
      </Col>
    </Row>
  </Container>
</Canvas>

### Fluid Container

You can use <b>&lt;Container fluid /&gt;</b> for width: 100% across all viewport and device sizes.

<Canvas>
  <Container fluid>
    <Row>
      <Col>
        <div>1 of 1</div>
      </Col>
    </Row>
  </Container>
</Canvas>

You can set breakpoints for the <b>fluid</b> prop. Setting it to a breakpoint (sm, md, lg, xl, xxl) will set the <b>Container</b> as fluid until the specified breakpoint.

<Canvas>
  <Container fluid="md">
    <Row>
      <Col>
        <div>1 of 1</div>
      </Col>
    </Row>
  </Container>
</Canvas>

## Auto-layout columns

When no column widths are specified the Col component will render equal width columns

### Equal-width

For example, here are two grid layouts that apply to every device and viewport, from xs to xl. Add any number of unit-less classes for each breakpoint you need and every column will be the same width.

<Canvas>
  <Container>
    <Row>
      <Col>
        <div>1 of 2</div>
      </Col>
      <Col>
        <div>2 of 2</div>
      </Col>
    </Row>
    <Row>
      <Col>
        <div>1 of 3</div>
      </Col>
      <Col>
        <div>1 of 3</div>
      </Col>
      <Col>
        <div>1 of 3</div>
      </Col>
    </Row>
  </Container>
</Canvas>

### Setting one column width

Auto-layout for flexbox grid columns also means you can set the width of one column and have the sibling columns automatically resize around it. You may use predefined grid classes (as shown below), grid mixins, or inline widths. Note that the other columns will resize no matter the width of the center column.

<Canvas>
  <Container>
    <Row>
      <Col>
        <div>1 of 3</div>
      </Col>
      <Col xs={6}>
        <div>2 of 3 (wider)</div>
      </Col>
      <Col>
        <div>3 of 3</div>
      </Col>
    </Row>
    <Row>
      <Col>
        <div>1 of 3</div>
      </Col>
      <Col xs={5}>
        <div>2 of 3 (wider)</div>
      </Col>
      <Col>
        <div>3 of 3</div>
      </Col>
    </Row>
  </Container>
</Canvas>

### Variable width content

Set the column value (for any breakpoint size) to "auto" to size columns based on the natural width of their content.

<Canvas>
  <Container>
    <Row className="justify-content-md-center">
      <Col xs lg="2">
        <div>1 of 3</div>
      </Col>
      <Col md="auto">
        <div>Variable width content</div>
      </Col>
      <Col xs lg="2">
        <div>3 of 3</div>
      </Col>
    </Row>
    <Row>
      <Col>
        <div>1 of 3</div>
      </Col>
      <Col md="auto">
        <div>Variable width content</div>
      </Col>
      <Col xs lg="2">
        <div>3 of 3</div>
      </Col>
    </Row>
  </Container>
</Canvas>

### Responsive grids

The Col lets you specify column widths across 6 breakpoint sizes (xs, sm, md, lg, xl and xxl). For every breakpoint, you can specify the amount of columns to span, or set the prop to <Col lg={true} /> for auto layout widths.

<Canvas>
  <Container>
    <Row>
      <Col sm="8">
        <div>sm=8</div>
      </Col>
      <Col sm="4">
        <div>sm=4</div>
      </Col>
    </Row>
    <Row>
      <Col sm>
        <div>sm=true</div>
      </Col>
      <Col sm>
        <div>sm=true</div>
      </Col>
      <Col sm>
        <div>sm=true</div>
      </Col>
    </Row>
  </Container>
</Canvas>

You can also mix and match breakpoints to create different grids depending on the screen size.

<Canvas>
  <Container>
    <Row>
      <Col xs={12} md={8}>
        <div>xs=12 md=8</div>
      </Col>
      <Col xs={6} md={4}>
        <div>xs=6 md=4</div>
      </Col>
    </Row>
    {/* Columns start at 50% wide on mobile and bump up to 33.3% wide on desktop */}
    <Row>
      <Col xs={6} md={4}>
        <div>xs=6 md=4</div>
      </Col>
      <Col xs={6} md={4}>
        <div>xs=6 md=4</div>
      </Col>
      <Col xs={6} md={4}>
        <div>xs=6 md=4</div>
      </Col>
    </Row>
    {/* Columns are always 50% wide, on mobile and desktop */}
    <Row>
      <Col xs={6}>
        <div>xs=6</div>
      </Col>
      <Col xs={6}>
        <div>xs=6</div>
      </Col>
    </Row>
  </Container>
</Canvas>
