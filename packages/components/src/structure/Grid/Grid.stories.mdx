# Grid System

Our grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive. Below is an example and an in-depth look at how the grid comes together.

## Container

Containers provide a means to center and horizontally pad your site’s contents. Use <b>Container</b> for a responsive pixel width.

<Canvas>
  <Container>
    <Row gutter="g-0">
      <Col>
        <div className="p-5 border bg-light">1 of 1</div>
      </Col>
    </Row>
  </Container>
</Canvas>

### Fluid Container

You can use <b>&lt;Container fluid /&gt;</b> for width: 100% across all viewport and device sizes.

<Canvas>
  <Container fluid>
    <Row gutter="g-0">
      <Col>
        <div className="p-5 border bg-light">1 of 1</div>
      </Col>
    </Row>
  </Container>
</Canvas>

You can set breakpoints for the <b>fluid</b> prop. Setting it to a breakpoint (sm, md, lg, xl, xxl) will set the <b>Container</b> as fluid until the specified breakpoint.

<Canvas>
  <Container fluid="md">
    <Row gutter="g-0">
      <Col>
        <div className="p-5 border bg-light">1 of 1</div>
      </Col>
    </Row>
  </Container>
</Canvas>

## Gutters

Gutters are the padding between your columns, used to responsively space and align content in the Bootstrap grid system.

### How they work

- Gutters are the gaps between column content, created by horizontal padding. We set padding-right and padding-left on each column, and use negative margin to offset that at the start and end of each row to align content.

- Gutters match our grid to the padding and margin spacing scale (see [Spacing page](/story/basic--spacing)).

- Gutters can be responsively adjusted. Use breakpoint-specific gutter classes to modify horizontal gutters, vertical gutters, and all gutters.

### How to use

To use custom gutters on a grid, add the prop gutter to the <b>&lt;Row /&gt;</b> and the values as class names. Example: <b>&lt;Row gutter="g-5" /&gt;</b>

By default, if no gutter prop is defined on the <b>&lt;Row /&gt;</b> element, our grid system will use a <b>pre-defined horizontal and vertical gutters</b> based on the [Layout](/story/basic-layout) specifications (<code>g-5 g-md-7 g-lg-8</code>).

See bellow examples for more details.
