TeaserCards are used in the Megamenu. Can be verticals or horizontals. For breakpoints XS ann SM, both variants have same size, but it differs in the texts.

### linkRenderer property

As the TeaserCard component can be consumed in several diferent systems - normally in the Megamenu, with diferent navigation methods, the links must be rendered in different ways. To add flexibility, the **linkRenderer** property will be used to render all links.

**IMPORTANT**: To have the correct look and fell, the label must be inside an "a tag".

#### Example 1 - "a" tag

```jsx
<TeaserCard
  ...
  linkRenderer={(link, label) => <a href={link}>{label}</a>}
  ...
/>
```

#### Example 2 - Link from router

```jsx
import Link from 'next/link'

<TeaserCard
  ...
  linkRenderer={(link, label) =>
    <Link href={link}>
        <a>{label}</a>
    </Link>
  }
  ...
/>
```
