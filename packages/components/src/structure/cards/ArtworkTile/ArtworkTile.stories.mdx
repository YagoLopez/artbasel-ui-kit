## Artworks Tile Card

These type of cards make up most of the interface and are key components for logically grouping page content and making it scalable.

### Room

When displayed in an OVR room the tile can be using a square, contained or masonary image type. The gallery name is always hidden.

### Listing

When displayed in the listing page, the image is displayed in masonary type and the gallery name is visible.

#### Masonry Image

  </ArtworkTile>
</Canvas>

### Catalogue

When used in catalogue page, the image is displayed in masonary type, the gallery name is visible and signpost labels are used underneath to indicate the show the artwork was displayed in.
