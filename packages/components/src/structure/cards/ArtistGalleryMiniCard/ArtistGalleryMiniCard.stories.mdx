## Behaviors

### Truncation of text

The title will get truncated using an ellipsis when passed two lines. Hovering over the ellipsis will show the full text using the HTML title attribute defined styling.

<Canvas>
  <Story
    name="Artist Gallery Mini Card Truncated text"
    args={{

      image: "https://picsum.photos/400",
      selectMode: {
        active: false,
        checked: false,
        onChange: () => { },
        disabled: false,
      },

</Canvas>
