## Cards Carousel

In some usages Carousel will need to extend itself beyound the container, to fill the viewport width.

In this case, we need to add an <code>overflow: 'hidden'</code> css style to the most top container, and additionally add a property <code>exceedTrack</code> to <code>Carousel</code> component.

Please check the source source (button Show Code) to see the complete code.

## Product Tile Carousel

The Carousel component can be used to display any kind of content (including cards or images) with pre-defined width.

In case of using with Product Tile Card, which is responsive and dont have a fixed width, the property <code>productTiles</code> can be used on the <code>Carousel.Slide</code> component to ajust the product tile width to be compatible with the Artist card.
